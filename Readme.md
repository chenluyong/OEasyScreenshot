
# 首页致词

这个项目，有幸得到你的浏览。

这是一个基于 Qt 开发的项目，

主要目的是完成一款接口极简的截图工具，即插即用。

在极简的基础上不断细化，添加一些细致的功能，为优质的软件添加一份助力。

一个优秀的项目，少不了各位的支持与参与，欢迎你加入我们。

# 关于项目

工程环境：Qt Creator 4.2.1

Qt库版本：Qt 5.8.1 （测试5.0以上都可以正常编译）
